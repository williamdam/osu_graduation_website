Cross-Platform Software Development: Web + iOS + Android

This page is a showcase of my versatility as a developer across multiple platforms. In this scenario, I built a cross-platform mobile application for iOS and Android, that allows the graduates to submit their photos and stories, directly to this page. The information is stored in a NoSQL database, where the Javascript on this page listens for changes, and displays the cards in 3 columns.

Technical Points: Flutter, Dart, iOS, Android, HTML, CSS, Javascript, DOM Manipulation, NoSQL (Firestore DB)

Lessons Learned: Cross-platform mobile application development and system design.
